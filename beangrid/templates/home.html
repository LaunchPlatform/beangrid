BeanGrid

Open-source plaintext based spreadsheet app

BeanGrid is a powerful spreadsheet application that processes plaintext data with Excel-like formulas. It supports YAML-based workbooks, cross-sheet references, and advanced formula evaluation.

Key Features:

  • 📊 Excel-like formula evaluation
  • 📁 YAML-based workbook format
  • 🔗 Cross-sheet cell references
  • 📈 Built-in functions (SUM, AVERAGE, MAX, MIN, COUNT, COUNTA, IF)
  • 🔄 Dependency tracking and circular reference detection
  • ⚡ Fast processing with dependency graphs
